# Compilació en C++

## clang:

```bash
clang++ -Wall -O2 -std=c++20 programa.cc -o programa.exe
```


## GCC:

```bash
g++ -Wall -O2 -std=c++20 programa.cc -o programa.exe
```


## Microsoft:

```bash
cl /EHsc /W4 /O2 /std:c++20 programa.cc
```

# Execució

En Windows `programa.exe`, en sistemes Unix `./programa.exe`.
